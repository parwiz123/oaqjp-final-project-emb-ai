# Emotion Detection Web Application (IBM AI Final Project)

## Project Name
oaqjp-final-project-emb-ai

## Overview
This project is a Flask-based Emotion Detection application that uses IBM Watson NLP Emotion Analysis API to detect emotions from user-provided text.

The system analyzes text and returns:
- anger
- disgust
- fear
- joy
- sadness
- dominant emotion

## Features
- Emotion detection using Watson NLP API
- Flask web server
- REST API endpoint
- Handles blank input validation
- Unit testing support
- Static code analysis (pylint)

## Technologies Used
- Python 3.10
- Flask
- IBM Watson NLP API
- Requests library
- Pylint
